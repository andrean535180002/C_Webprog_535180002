Pengertian Web dan Programming

Web programming terdiri dari dua suku kata yaitu web dan programming.

Programming dapat diartikan sebagai proses, cara, atau pembuatan suatu program.

Web dapat diartikan sebagai jaringan komputer yang terdiri dari kumpulan situs internet yang menawarkan teks, grafik, suara dan sumber daya animasi melalui protokol transfer hypertext (HTTP).
